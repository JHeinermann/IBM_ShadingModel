DATA PREPARATION

To initialize our Stand-Model, we need data of a forest stand or rather of individual trees. The data required includes:

  • Tree position
    • X-Position [m]
    • Y-Position [m]
  • DBH (Diameter at Breast Height) [cm]

We used data that was obtained during the VERMOS project. From this stand-data we created 10 artificial stands using the R-Package shar.
The R-Script of Artificial Stand Creation can be found HERE.
We modified some functions of shar which can be downloaded HERE.
